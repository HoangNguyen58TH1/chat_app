# Rails WebSocket Chat Application with ActionCable

This repository demonstrates a real-time chat application implementation using WebSocket technology through Rails ActionCable. The application showcases both backend and frontend integration of WebSocket functionality.

## Technology Stack

### Backend

- Ruby on Rails
- ActionCable (Rails' WebSocket framework)
- SQLite3 (Database)

### Frontend

- JavaScript
- ActionCable JavaScript client
  - `consumer.js`: Creates WebSocket connection using `createConsumer()` from `@rails/actioncable`
  - `chat_channel.js`: Handles real-time message subscription and DOM updates
  - `index.js`: Entry point for channel subscriptions
- HTML/CSS

## Key Components

### Backend Implementation

1. **ActionCable Setup**

   - Located in `app/channels/application_cable/`
   - `connection.rb`: Base class for WebSocket connections
     - Inherits from `ActionCable::Connection::Base`
     - Handles the initial WebSocket connection setup
   - `channel.rb`: Base class for all channels
     - Inherits from `ActionCable::Channel::Base`
     - Provides common functionality for all channel classes
   - These files are automatically generated by Rails when setting up ActionCable
   - They serve as the foundation for WebSocket communication in the application
2. **Chat Channel**

   - Located in `app/channels/chat_channel.rb`
   - Handles real-time message broadcasting
   - Manages WebSocket connections for chat functionality
3. **Models**

   - Message model for storing chat messages
4. **Controllers**

   - RESTful controllers for handling HTTP requests
   - ActionCable channel controllers for WebSocket communication

### Frontend Implementation

1. **JavaScript Integration**

   - ActionCable JavaScript client setup
   - WebSocket connection management
   - Real-time message handling
2. **Channel Subscriptions**

   - Client-side channel subscription
   - Message broadcasting and receiving
   - Connection state management

## WebSocket Implementation Details

### Backend (ActionCable)

1. **Connection Setup**

   ```ruby
   # app/channels/application_cable/connection.rb
   module ApplicationCable
     class Connection < ActionCable::Connection::Base
       # Connection authentication and setup
     end
   end
   ```
2. **Channel Definition**

   ```ruby
   # app/channels/chat_channel.rb
   class ChatChannel < ApplicationCable::Channel
     def subscribed
       # Channel subscription logic
     end

     def receive(data)
       # Message handling logic
     end
   end
   ```

### Frontend (JavaScript)

1. **WebSocket Connection**
   ```javascript
   // app/javascript/channels/chat.js
   import consumer from "./consumer"

   consumer.subscriptions.create("ChatChannel", {
     connected() {
       // Connection established
     },
     received(data) {
       // Handle received messages
     }
   });
   ```

## Setup and Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   bundle install
   yarn install
   ```
3. Setup database:
   ```bash
   rails db:create db:migrate
   ```
4. Start the server:
   ```bash
   rails server
   ```
